### superordinate_list
list all persistent superordinates of the current context, including all descendants (children, grandchildren, etc.) shown as an indented tree.
args: none
example:
~~~json
{
  "thoughts": ["I want to see my current superordinates and their full tree."],
  "headline": "Listing superordinates",
  "tool_name": "superordinate_list",
  "tool_args": {}
}
~~~
